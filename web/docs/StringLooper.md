# String Append Looper

Repeats a given string a specified number of times.

## Overview

This utility node takes an input string and an integer, `loops`. It concatenates the input string to itself `loops` number of times. The final output string will contain `loops + 1` total copies of the original text. For example, if the input is "A" and `loops` is 2, the output will be "AAA".

## Parameters

- **text**: The string to be repeated.
- **loops**: The number of times to append the string to the original.

## Outputs

- **looped_text**: The resulting string after repetition.
